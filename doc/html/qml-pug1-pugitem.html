    PugItem

    The base class of all classes in the Pug module. More...

    PugItem instantiates the C++ class PugItem

    Detailed Description

    PugItem is the base class of all classes in the Pug module. It has properties and methods for instance naming, child Pug items, the parent Pug item and logging that are available to all Pug items.
