    Element instantiates the C++ class Element

    Detailed Description

    Elements encapsulate the path of a directory, a file, or a sequence of files, and related data. BranchBase-derived classes contain a list of elements, each which is an instance of Element. Elements have convenience read-only properties that return the directory, baseName, frameSpec and extension of their path. They also store a FrameList object and corresponding read-only frames convenience property.
